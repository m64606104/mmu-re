# 语音识别功能更新说明

## 🎉 主要更新

已将语音识别功能从浏览器内置的 Web Speech API 更换为基于 **智谱清言 API** 的云端识别方案。

## ✨ 新功能特点

### 1. 可配置的语音识别
- 在设置页面添加了"语音转文字"配置区域
- 支持开关控制，未启用时使用"录音+手动输入"模式
- 启用后可配置自定义 API 地址、API Key 和识别模型

### 2. 智能识别流程
```
用户点击语音按钮 → 开始录音 → 停止录音 → 
  ↓
如果启用了语音识别？
  ├─ 是 → 自动调用 API 识别 → 显示识别结果
  └─ 否 → 直接显示手动输入框
```

### 3. 推荐配置
**智谱清言 glm-4-flash**
- ✅ 免费无限次数
- ✅ 识别准确度高  
- ✅ 支持60秒语音
- ✅ 响应速度快

**配置信息：**
- API地址: `https://open.bigmodel.cn/api/paas/v4`
- 模型名称: `glm-4-flash`
- 获取API Key: https://open.bigmodel.cn

## 📝 使用方法

### 第一步：配置语音识别（可选）

1. 进入"设置"页面
2. 找到"🎤 语音转文字"部分
3. 打开"启用语音识别"开关
4. 填写以下信息：
   - API 地址
   - API Key
   - 识别模型名称
5. 点击"保存配置"

### 第二步：使用语音功能

1. **启用了语音识别的情况：**
   - 点击麦克风按钮开始录音
   - 说出你想发送的内容
   - 点击停止录音
   - 系统自动识别并显示文字
   - 确认或修改后发送

2. **未启用语音识别的情况：**
   - 点击麦克风按钮开始录音
   - 说出你想发送的内容
   - 点击停止录音
   - 手动输入语音内容
   - 发送（语音文件会一起发送）

## 🔧 技术实现

### 新增文件

#### `src/utils/speechToText.ts`
语音转文字工具函数，包含：
- `transcribeAudio()` - 调用智谱清言 API 进行语音识别
- `isValidSpeechConfig()` - 检查配置是否有效
- `audioToBase64()` - 音频格式转换

### 修改文件

#### `src/types.ts`
扩展 `ApiConfig` 接口，添加语音识别配置：
```typescript
speechToText?: {
  enabled: boolean;
  apiUrl?: string;
  apiKey?: string;
  model?: string;
}
```

#### `src/components/SettingsScreen.tsx`
添加语音转文字配置UI：
- 开关控制
- API 配置表单
- 推荐配置提示

#### `src/components/ChatScreen.tsx`
简化语音功能实现：
- 移除 Web Speech API 相关代码
- 直接录音并保存为文件
- 在 `mediaRecorder.onstop` 回调中触发识别
- 根据配置决定是否自动识别

## 🎯 优势对比

| 特性 | Web Speech API (旧方案) | 智谱清言 API (新方案) |
|------|------------------------|---------------------|
| 浏览器兼容性 | 仅 Chrome/Edge | 所有浏览器 |
| 网络依赖 | Google 服务（国内受限） | 可自定义 |
| 识别准确度 | 中等 | 高 |
| 可控性 | 低 | 高 |
| 配置灵活性 | 无 | 完全可配置 |
| 离线使用 | 不支持 | 不支持 |
| 免费额度 | 无限 | 根据API提供商 |

## 💡 使用建议

1. **推荐场景：**
   - 智谱清言 glm-4-flash 适合大多数用户（免费且准确）
   - 可以使用其他兼容 OpenAI 格式的语音识别 API

2. **注意事项：**
   - 录音时长建议控制在 60 秒以内
   - 确保网络连接稳定
   - API Key 请妥善保管

3. **故障排除：**
   - 如果识别失败，可以手动输入内容
   - 检查 API 配置是否正确
   - 确认 API Key 是否有效且有足够额度

## 🔄 迁移指南

如果你之前使用的是旧版本：

1. 无需任何操作即可使用"录音+手动输入"模式
2. 如需启用自动识别，请按照上述配置步骤操作
3. 旧的 Web Speech API 代码已完全移除
