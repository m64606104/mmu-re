# 群聊私聊记忆连贯性功能

## 🎯 功能概述

解决了AI在群聊中说"我私聊你"后，用户进入私聊时AI不记得群聊对话内容的问题。

## 🌉 工作原理

### 1. 意图检测
当AI在群聊中发送包含私聊意图的消息时，系统会自动检测：
- "我私聊你"
- "那你私聊我"
- "私下聊"
- "我马上私聊你"
- "我给你发个大红包"（在私聊语境中）

### 2. 记忆桥接创建
检测到私聊意图后，系统会：
- 保存群聊的最近10条消息作为上下文
- 记录AI的私聊意图消息
- 创建24小时有效的记忆桥接

### 3. 私聊上下文注入
当用户进入私聊时，系统会：
- 检查是否存在相关的群聊记忆桥接
- 将群聊对话背景注入到AI的系统提示词中
- AI基于群聊背景进行连贯回复

## 🔧 技术实现

### 核心文件
- `/src/utils/groupToPrivateMemoryService.ts` - 记忆桥接服务
- `/src/utils/groupToPrivateMemoryTest.ts` - 测试工具
- `/src/components/ChatScreen.tsx` - 集成点

### 数据结构
```typescript
interface MemoryBridge {
  id: string;
  type: 'group_to_private' | 'private_to_group';
  sourceId: string;
  targetId: string;
  userId: string;
  bridgeMessage: string;
  contextSummary: string;
  createdAt: number;
  expiresAt: number; // 24小时后过期
}
```

### 集成点
1. **群聊AI回复生成时** - 检测私聊意图
2. **私聊AI回复生成前** - 注入群聊上下文

## 📱 使用场景

### 场景1: 工作压力讨论
**群聊对话**:
```
用户: 最近项目进展不太顺利，我有点焦虑
AI: 别担心，我理解你的感受。要不我私聊你，我们详细聊聊具体情况？
用户: 好的，谢谢理解
```

**私聊对话**（AI会记得群聊内容）:
```
用户: 我来了
AI: 基于你在群聊提到的项目进展问题，我想说：别给自己太大压力，每个项目都会有困难时期。我们可以一起分析具体的问题...
```

### 场景2: 红包发放
**群聊对话**:
```
用户: 最近手头有点紧
AI: 那你私聊我，我给你发个大红包
用户: 真的吗？谢谢！
```

**私聊对话**:
```
用户: 我来领红包啦
AI: 记得我们在群聊说的，你最近手头有点紧。这个红包希望能帮到你，记得好好规划使用哦~
```

## ⚙️ 配置选项

### 过期时间
- 默认：24小时
- 可在 `groupToPrivateMemoryService.ts` 中修改 `CONTEXT_EXPIRE_TIME`

### 意图关键词
- 在 `detectPrivateChatIntent()` 方法中配置
- 支持添加新的私聊意图关键词

### 上下文消息数量
- 默认保存最近10条群聊消息
- 可在 `createPrivateChatIntent()` 中调整

## 🚀 性能优化

### 自动清理
- 24小时后自动清理过期数据
- 每次加载时执行清理

### 内存管理
- 只保存必要的上下文消息
- 限制桥接数据大小

### 懒加载
- 只在私聊中才检查群聊上下文
- 避免不必要的计算

## 🐛 常见问题

### Q: AI没有检测到私聊意图
A: 检查关键词配置，确保消息包含私聊意图关键词

### Q: 私聊中没有群聊上下文
A: 确认群聊和私聊的ID匹配，且用户ID一致

### Q: 上下文信息不准确
A: 检查消息时间戳和上下文消息数量配置
