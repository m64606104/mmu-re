# 智能HTML模块生成系统

## 🎨 设计理念

参考**【🍊】中插随机html**的设计思想：
- **创意多样化**：不局限于固定格式，根据场景自由生成
- **内容真实丰富**：必须有具体内容，不是空模板
- **与对话紧密结合**：根据角色状态和对话内容生成
- **视觉美观**：排版精美，有风格感

## 📱 已实现的模块类型

### **1. 搜索记录（SearchHistoryView）** 🆕

模拟浏览器搜索历史界面，展示用户最近搜索的内容。

**格式：**
```
开始搜索记录[
搜索记录[关键词|时间|设备|详情说明]
搜索记录[关键词|时间|设备|详情说明]
...
]结束搜索记录
```

**示例：**
```
开始搜索记录[
搜索记录[如何学习CSS|2025-11-10 14:20|桌面浏览器|搜索了关于CSS选择器和Flexbox布局的核心知识。]
搜索记录[原生JS实现动画|2025-11-09 18:45|移动设备|查询了使用 requestAnimationFrame 来创建平滑的网页动画效果的方法。]
搜索记录[HTML5语义化标签|2025-11-09 09:10|桌面浏览器|了解 <article>, <section>, <nav> 等标签的正确使用场景。]
]结束搜索记录
```

**界面特点：**
- 列表视图：显示所有搜索记录
- 详情面板：点击可滑动查看详情
- 时间设备：显示搜索时间和设备类型
- 交互流畅：CSS transition动画

**使用场景：**
- 用户问："你最近在学什么？" → 发搜索记录
- 用户问："有什么推荐吗？" → 发相关搜索记录
- 想暗示自己最近的关注点 → 发搜索记录
- 分享查到的资料 → 发搜索记录

## 💻 技术架构

### **统一管理：SmartHTMLGenerator**

```typescript
// 检测内容类型
const htmlType = SmartHTMLGenerator.detectHTMLType(content);
// 返回: 'search-history' | 'xiaohongshu' | 'zhihu' | 'weibo' | null

// 获取完整Prompt
const prompt = SmartHTMLGenerator.getFullPrompt();
```

### **组件渲染流程**

```typescript
// ChatScreen.tsx
if (htmlType) {
  // 不拆分消息，保持完整
  socialFeed: {
    platform: htmlType,
    rawContent: assistantMessage  // 完整内容
  }
}

// 渲染对应组件
{message.socialFeed?.platform === 'search-history' && (
  <SearchHistoryView rawContent={message.socialFeed.rawContent} />
)}
```

### **解析流程**

```typescript
// SearchHistoryView.tsx
const parseContent = (): SearchRecord[] => {
  const records: SearchRecord[] = [];
  const regex = /搜索记录\[(.*?)\|(.*?)\|(.*?)\|([\s\S]*?)\]/g;
  
  while ((match = regex.exec(rawContent)) !== null) {
    records.push({
      keyword: match[1],
      time: match[2],
      device: match[3],
      details: match[4]
    });
  }
  
  return records;
};
```

## 🚀 扩展性设计

### **添加新模块类型非常简单**

#### **1. 创建组件**
```typescript
// src/components/NewModuleView.tsx
const NewModuleView: React.FC<{ rawContent: string }> = ({ rawContent }) => {
  // 解析和渲染逻辑
  return <div>...</div>;
};
```

#### **2. 更新类型定义**
```typescript
// src/types.ts
export interface SocialFeedMessage {
  platform: '... | 'new-module';
  rawContent: string;
}
```

#### **3. 添加检测逻辑**
```typescript
// src/utils/smartHTMLGenerator.ts
static detectHTMLType(content: string) {
  if (content.includes('新模块标记[')) {
    return 'new-module';
  }
  // ...
}
```

#### **4. 添加Prompt**
```typescript
static getNewModulePrompt(): string {
  return `新模块生成规则...`;
}
```

#### **5. 集成渲染**
```typescript
// src/components/ChatScreen.tsx
{message.socialFeed?.platform === 'new-module' && (
  <NewModuleView rawContent={message.socialFeed.rawContent} />
)}
```

## ⚠️ 生成规则

### **必须遵守的规则**

1. **内容完整真实**
   - ❌ 禁止：空模板、填充符号、Lorem Ipsum
   - ✅ 必须：具体文字、真实数据、完整内容

2. **与对话相关**
   - 根据当前话题生成
   - 符合角色身份和性格
   - 自然融入对话流程

3. **视觉美观**
   - 宽度不超过310px
   - 排版清晰美观
   - 有视觉风格

4. **数据真实**
   - 时间合理（最近的记录）
   - 数字合理（点赞、评论数）
   - 设备真实（桌面/移动）
