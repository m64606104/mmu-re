# 新建对话功能更新说明

## 🎉 最新功能更新

**修复代码的过程顺手更新了几个新功能：**

- 📱 **子对话增强** - 现在可以随意放大缩小和移动子对话窗口，支持触摸和鼠标操作
- 🎵 **听音乐功能** - 初步构建了AI音乐分享功能：
  - 动态歌词获取 (API + 手动输入)  
  - AI实时感知播放状态和歌词进度
  - 像真人朋友一样讨论音乐和歌词
- 🔍 **聊天记录查找** - 聊天记录查找和筛选功能，快速定位历史消息
- 📋 **消息转发** - 可以转发和合并转发聊天记录，AI会智能识别并理解聊天记录的内容
- ✨ **其他优化** - 还有一些细节改进和用户体验优化

---

## 更新内容

本次更新重构了新建对话的流程，将"添加好友"和"发起群聊"分别独立为两个专门的页面，提供更细致的配置选项。

## 新增功能

### 1. 添加好友页面 (AddFriendScreen)

**功能特点：**
- 完整的角色设置界面
- 包含以下配置项：
  - 角色头像（支持上传）
  - 备注名（必填）
  - 角色网名
  - 人物设定
  - 性格特征
  - 语言风格
  - 语言示例
- **不显示记忆库**（符合需求）
- 点击"确认添加"后：
  - 自动创建私聊对话
  - 进入聊天页面
  - 显示系统提示："你已成功添加好友 [角色名]，可以开始聊天了"

### 2. 发起群聊页面 (CreateGroupScreen)

**功能特点：**
- 自定义群头像（支持上传）
- 自定义群名称（必填）
- 自定义群备注名（可选，仅自己可见）
- 添加群成员功能：
  - 可以从已有联系人列表中选择
  - 支持多选
  - 显示已选成员列表
  - 可以移除已选成员
- 点击"创建群聊"后：
  - 自动创建群聊对话
  - 进入群聊页面
  - 为每个加入的成员显示系统提示："[角色名] 已加入群聊"

### 3. 新建对话选择页面 (NewConversationScreen)

**功能更新：**
- 重新设计为导航选择器
- 两个大按钮：
  - **添加好友**：进入添加好友页面
  - **发起群聊**：进入发起群聊页面
- 提供清晰的功能说明

## 用户流程

### 添加好友流程
1. 点击"新建对话"按钮
2. 选择"添加好友"
3. 填写角色信息（备注名必填，其他可选）
4. 点击"确认添加"
5. 自动跳转到聊天页面，显示成功提示

### 发起群聊流程
1. 点击"新建对话"按钮
2. 选择"发起群聊"
3. 设置群头像、群名称、群备注名
4. 点击"添加成员"选择联系人
5. 点击"创建群聊"
6. 自动跳转到群聊页面，显示所有成员加入提示

## 系统消息显示

- 系统消息在聊天界面中以灰色居中小标签形式显示
- 添加好友：显示 "你已成功添加好友 [角色名]，可以开始聊天了"
- 群聊成员：每个成员显示 "[角色名] 已加入群聊"

## 注意事项

1. 添加好友页面不显示记忆库管理功能
2. 群备注名仅用户自己可见，不影响群名称
3. 发起群聊时至少需要选择一个群成员
4. 只能从已有的私聊联系人中选择群成员
5. 系统消息不可删除，作为对话历史的一部分
