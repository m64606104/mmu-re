# 🌟 智能财务系统 - 真实大世界生态

## ✅ **核心功能**

### 1. 🤖 **智能职业识别**
系统通过AI分析角色设置，自动识别AI的职业类型并配置相应的收入：

#### 支持的职业类型：
- **上班族（office_worker）**：月中发工资，可能有奖金和补贴
  - 示例：程序员、设计师、销售、财务等
  - 发薪日：每月15号左右
  - 收入：基础月薪 + 绩效奖金 + 餐补/交通补贴

- **学生（student）**：月初发生活费，可能半工半读
  - 示例：大学生、研究生
  - 发薪日：每月1号
  - 收入：生活费 + 可选兼职收入（每周）

- **自由职业者（freelancer）**：收入不定期
  - 示例：独立设计师、作家、咨询师
  - 频率：不定期（1-7天随机）
  - 收入：项目收入、稿费等

- **创业者/老板（entrepreneur）**：收入较高且不定期
  - 示例：创业公司老板、个体户
  - 频率：不定期
  - 收入：利润分成、营收等

- **艺术家/创作者（artist）**：多样化收入来源
  - 示例：画家、音乐人、UP主
  - 频率：不定期
  - 收入：稿费、演出费、打赏等

- **服务业（service）**：高频结算
  - 示例：服务员、快递员、外卖员
  - 频率：每天或每周
  - 收入：工资 + 提成

### 4. 🔄 **自动收入发放**

#### **定时检查**
- App启动时：立即检查一次
- 运行中：每小时检查一次
- 检查逻辑：对比当前时间和`nextIncomeTime`

#### **发放流程**
```typescript
// 每小时执行一次
setInterval(async () => {
  await processAllAutoIncome();
}, 60 * 60 * 1000);

// 处理逻辑
for (const config of incomeConfigs) {
  if (now >= config.nextIncomeTime) {
    // 计算金额（支持随机范围）
    let amount = config.baseAmount;
    if (config.randomRange) {
      amount = random(config.randomRange[0], config.randomRange[1]);
    }
    
    // 发放收入
    await addTransaction(aiId, 'income', amount, ...);
    
    // 更新下次收入时间
    config.nextIncomeTime = now + calculateDelay(config.frequency);
  }
}
```
