### 3. AI自动发朋友圈 🤖

**功能说明：**
AI角色会根据自己的性格和设定，自动发布朋友圈动态！

**触发机制：**
- ⏰ 每10分钟检查一次
- 📅 时间间隔：2-12小时
- 🎲 触发概率：60%
- 🎯 只对启用功能的私聊角色生效

**内容生成：**
- 基于角色性格和说话风格
- 可能的内容类型：
  - 日常分享
  - 心情感悟
  - 有趣见闻
  - 美食美景
  - 生活碎片
- 自动添加emoji表情
- 长度：1-3句话，50-150字

**示例：**
```
今天天气超好！☀️ 去公园散步看到好多小狗狗，好想养一只~

刚看完一部超好看的电影，感动到哭😭 强烈推荐！

周末在家做了新菜，味道还不错哦~ 下次做给你尝尝😋
```

### 4. 朋友圈智能互动 💬

**功能说明：**
AI角色会自动给你的朋友圈点赞和评论！

**触发机制：**
- ⏰ 每3分钟检查一次
- 👍 点赞概率：50%
- 💬 评论概率：30%
- 🎯 只对用户发布的朋友圈生效

**点赞功能：**
- 自动检测新动态
- 随机决定是否点赞
- 不会重复点赞

**评论功能：**
- AI理解动态内容
- 生成符合角色性格的评论
- 评论类型：
  - 点赞认同
  - 情感共鸣
  - 幽默调侃
  - 关心问候
  - 好奇提问
- 不会重复评论

**示例评论：**
```
你的朋友圈：今天加班到好晚，累死了😫
AI评论：辛苦啦！早点休息~ 明天会更好的💪

你的朋友圈：周末去爬山，风景超美！🏔️
AI评论：哇好羡慕！下次带我一起去呀~

你的朋友圈：新买的蛋糕好好吃😋
AI评论：看起来就很好吃！是什么口味的呀？
```

## 🎯 使用指南

### 第一步：设置个人资料
1. 进入**社交**页面
2. 点击**我的资料**（蓝色图标）
3. 点击**编辑资料**按钮
4. 上传头像和封面图
5. 设置用户名和个性签名
6. 点击**保存**

### 第二步：发布朋友圈
1. 在资料页面点击**朋友圈**
2. 点击右上角**相机**图标
3. 输入文字内容
4. （可选）点击**添加图片**上传照片
5. 点击**发布**

### 第三步：启用AI功能
1. 进入任意私聊对话
2. 点击右上角**功能**按钮
3. 选择以下功能：
   - ✅ **AI自动发朋友圈**
   - ✅ **朋友圈智能互动**
4. 点击**保存**

### 第四步：体验AI互动
- **AI发朋友圈**：等待2-12小时，AI会自动发布动态
- **AI互动**：你发朋友圈后，AI会在3分钟内点赞或评论

## 📊 功能配置

### AI自动发朋友圈配置
```typescript
{
  enabled: true,           // 是否启用
  minInterval: 2,          // 最小间隔2小时
  maxInterval: 12,         // 最大间隔12小时
  triggerProbability: 0.6  // 60%触发概率
}
```

### 朋友圈智能互动配置
```typescript
{
  likeProb: 0.5,      // 50%点赞概率
  commentProb: 0.3,   // 30%评论概率
  checkInterval: 3    // 每3分钟检查
}
```

## 🔄 工作流程

### 用户发布朋友圈
```
用户输入内容 → 上传图片 → 点击发布
    ↓
保存到moments数组
    ↓
显示在朋友圈列表
    ↓
AI检测到新动态（3分钟内）
    ↓
AI决定是否点赞/评论
    ↓
生成评论内容
    ↓
添加到动态的评论列表
```

### AI自动发朋友圈
```
定时器触发（每10分钟）
    ↓
检查启用功能的角色
    ↓
判断时间间隔和概率
    ↓
调用API生成内容
    ↓
创建朋友圈动态
    ↓
添加到moments列表
    ↓
记录发布时间
```

### 朋友圈内容生成
1. 读取角色设定（性格、风格）

### 评论内容生成
1. 读取用户朋友圈内容
2. 结合角色性格
3. 理解动态语境
4. 生成符合角色的评论
5. 简短、自然、有趣
