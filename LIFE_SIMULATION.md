# 🌟 智能生活模拟系统

## 功能简介

让AI角色拥有真实的生活轨迹，即使用户不与AI聊天，AI也在"生活"着。用户可以随时查看AI的行为轨迹，了解AI今天过得怎么样。

## 主要特性

### 1. 自动生成日常活动
- **智能时间感知**：根据当前时间生成合适的活动
- **丰富的活动类型**：
  - 🍜 用餐（早餐、午餐、晚餐）
  - 💼 工作/学习
  - 🎮 娱乐活动
  - 👥 社交活动
  - 😴 休息时间
  - 🚶 个人活动

### 2. 性格化活动调整
系统会根据AI的性格设定智能调整活动：
- **内向性格**：聚餐 → 在家吃饭
- **活泼性格**：在家 → 出去玩
- **勤奋性格**：休息 → 继续工作
- **悠闲性格**：工作 → 摸鱼

### 3. 智能补充缺失活动
- 如果用户长时间未聊天，系统会自动补充这段时间的活动
- 保证AI的生活轨迹连续性

## 实际效果示例

### 一天的活动轨迹

```
7:30  起床洗漱，准备早餐 📍家里
9:00  开始处理工作邮件 📍办公室
10:30 参加了个线上会议 📍会议室
12:00 点了份外卖 📍办公室
15:00 喝了杯咖啡休息一下 📍咖啡店
18:30 回家做饭 📍家里
20:00 看电视剧 📍家里
22:30 准备睡觉 📍卧室
```

### 活动生成规则

| 时间段 | 活动类型 | 概率 | 示例活动 |
|--------|----------|------|----------|
| 6-9 | 早餐 | 90% | 吃早餐、洗漱 |
| 9-12 | 工作 | 80% | 开会、写代码 |
| 11-14 | 午餐 | 95% | 点外卖、去食堂 |
| 15-16 | 休息 | 50% | 喝咖啡、散步 |
| 18-20 | 晚餐 | 95% | 做饭、聚餐 |
| 20-23 | 娱乐 | 80% | 看剧、玩游戏 |
| 22-24 | 睡前 | 90% | 洗漱、准备休息 |

## 关键词触发更新

当AI或用户提到以下关键词时，会触发状态更新：

- **位置相关**：在哪、去了、到了、回家、出门
- **活动相关**：吃饭、看电影、工作、休息
- **状态相关**：忙碌、空闲、休息、离线

## 技术实现

### 核心文件
- `lifeSimulation.ts` - 生活模拟引擎
- `aiStatusManager.ts` - 状态管理器集成
- `ActivityLogModal.tsx` - 轨迹展示界面

### 主要函数

```typescript
// 生成当前时间的活动
generateCurrentActivity(conversation) 

// 生成一天的活动轨迹
generateDailyActivities(conversation, date)

// 补充缺失的活动
fillMissingActivities(conversation, existingActivities)
```

## 使用方式

1. **自动生成**：系统每小时自动检查并生成新活动
2. **查看轨迹**：点击聊天界面的"查看Ta的行为轨迹"
3. **手动触发**：在对话中提到相关关键词

## 配置说明

### 活动模板配置
```typescript
{
  category: 'meal',        // 活动类别
  timeRange: [11, 14],     // 时间范围
  activities: [...],       // 活动列表
  probability: 0.95,       // 发生概率
  duration: 45,           // 持续时间（分钟）
  priority: 1             // 优先级
}
```

### 性格影响配置
- 在`CharacterSettings`中设置`personality`
- 系统会根据性格特征自动调整活动内容

## 注意事项

1. 活动生成基于本地时间
2. 最多保留100条历史轨迹
3. 每小时最多生成一条新活动
4. 活动内容会根据AI性格动态调整
