### 2. QQ空间风格内容生成器

**新增文件**: `src/utils/qqSpaceStyleGenerator.ts`

#### 📱 实现的5种经典格式

##### 1. 转发说说（纯文字）
**样式参考**: 图5
- 灰色引用框
- "xxx说：" 格式
- 真实网络用语
- 适合：活泼型AI、中午休息时间

**示例**:
```
哈哈哈哈哈哈哈哈

一位网友说：用户物底操了：刚才看到有人说孙悟空全是糕潮时喊的是"俺老孙去也"
```

##### 2. 小说/文段截图
**样式参考**: 图2
- 长段落纯文字
- 情感化内容
- 文艺范儿
- 适合：文艺型AI、深夜时段

**示例**:
```
这段很萌...两个人互相取暖......

他拿心下的春帆跃散了，那是痛苦的尖布下，是缱绻，是依恋...
（长文段内容）
```

##### 3. 教程/素材分享
**样式参考**: 图3
- 多图展示（2-4张）
- 详细步骤说明
- 实用性强
- 适合：知识型AI

**示例**:
```
领子教学：有关衣服领子的绘制画法参考教程，很实用，特需学习呢~

[教程图片1] [教程图片2] [教程图片3]
```

##### 4. 游戏截图
**样式参考**: 图1、图4
- 游戏画面（1-3张）
- 游戏吐槽文字
- 真实玩家口吻
- 适合：活泼型AI、晚上时段

**示例**:
```
我要哭了这个结尾到底怎么打

[游戏截图1] [游戏截图2] [游戏截图3]
```

##### 5. 音乐分享
- 经典歌曲推荐
- 简短评论
- 情感表达

**示例**:
```
单曲循环中...

🎵 起风了 - 买辣椒也用券
```

### 3. 集成到朋友圈系统

**修改文件**: `src/utils/diverseMomentsGenerator.ts`

#### 格式权重分配

**总体分布**:
```typescript
原有格式（75%）:
- 音乐分享: 15%
- 公众号文章: 10%
- 单图: 10%
- 多图: 8%
- 生活吐槽: 8%
- 优惠券: 6%
- 纯文字: 5%
- 新闻分享: 4%
- 微博分享: 4%
- 大事件: 3%
- 心情检查: 2%

QQ空间格式（25%）:
- 转发说说（纯文字）: 10%
- 转发说说（带图）: 5%
- 小说文段: 4%
- 教程分享: 3%
- 游戏截图: 3%
```

#### 智能推荐机制

**根据AI性格**:
```typescript
活泼型AI:
- 游戏截图 × 2.0
- 转发说说 × 1.5

文艺型AI:
- 小说文段 × 2.5

知识型AI:
- 教程分享 × 2.0
```

**根据时间段**:
```typescript
深夜/清晨 (22:00-06:00):
- 小说文段 × 1.8

中午休息 (12:00-14:00):
- 转发说说 × 1.3

晚上 (18:00-21:00):
- 游戏截图 × 1.5
```

#### 格式详细配置

**添加的case分支**:
```typescript
case 'qq_forward_text':
  imageCount = 0;
  textLength = 'medium';
  contentStyle = 'casual';
  hasHashtags = false;
  break;

case 'qq_forward_image':
  imageCount = 1-3张;
  textLength = 'short';
  contentStyle = 'casual';
  hasHashtags = false;
  break;

case 'qq_novel_text':
  imageCount = 0;
  textLength = 'long';  // 小说文段通常很长
  contentStyle = 'emotional';
  hasHashtags = false;
  break;

case 'qq_tutorial':
  imageCount = 2-4张教程图;
  textLength = 'short';
  contentStyle = 'informative';
  hasHashtags = false;
  break;

case 'qq_game_screenshot':
  imageCount = 1-3张游戏截图;
  textLength = 'short';
  contentStyle = 'emotional';
  hasHashtags = true;
  break;
```

#### 内容生成方法

**添加的生成函数**:
1. `generateQQForwardContent()` - 转发说说
2. `generateQQNovelTextContent()` - 小说文段
3. `generateQQTutorialContent()` - 教程分享
4. `generateQQGameContent()` - 游戏截图
