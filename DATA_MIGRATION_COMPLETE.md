## 🔄 使用流程

### 导出数据

1. **打开设置页面**
   - 点击主屏幕的"设置"图标
   - 或从其他页面进入设置

2. **找到数据管理模块**
   - 滚动到页面底部
   - 找到"数据管理"区域

3. **点击导出按钮**
   - 点击绿色的"导出全部数据"按钮
   - 系统自动收集所有数据

4. **查看导出统计**
   - 弹窗显示详细的数据统计
   - 包括对话数、消息数、文档数等

5. **保存文件**
   - 文件自动下载到下载文件夹
   - 文件名格式：`momoyu_全数据备份_YYYY-MM-DD.json`

### 导入数据

1. **准备备份文件**
   - 确保有之前导出的JSON文件
   - 文件格式必须正确

2. **点击导入按钮**
   - 点击蓝色的"导入全部数据"按钮
   - 系统打开文件选择器

3. **选择备份文件**
   - 选择要导入的JSON文件
   - 系统自动读取文件内容

4. **查看导入预览**
   - 弹窗显示备份文件的详细信息
   - 包括备份时间和数据统计

5. **确认导入**
   - ⚠️ 警告：这将覆盖当前所有数据！
   - 建议先导出当前数据作为备份
   - 确认后开始导入

6. **自动刷新**
   - 导入成功后显示统计信息
   - 页面自动刷新应用新数据

## 📊 导出文件格式

### 文件结构

```json
{
  "exportDate": "2025-11-11T14:30:00.000Z",
  "appVersion": "1.0.0",
  "dataType": "full-backup",
  "stats": {
    "conversations": 10,
    "messages": 523,
    "moments": 45,
    "contacts": 8,
    "documents": 12,
    "images": 3,
    "profiles": 10,
    "relationships": 15
  },
  "data": {
    "conversations": [...],
    "contacts": [...],
    "documentLibrary": [...],
    "userProfile": {...},
    "apiConfig": {...},
    "moments_xxxxx": {...},
    "relationships": [...],
    "landscapeImage": "...",
    "bannerImage": "...",
    ... // 所有其他localStorage数据
  }
}
```

### 字段说明

- **exportDate**: 导出时间（ISO 8601格式）
- **appVersion**: 应用版本号
- **dataType**: 数据类型标识（full-backup）
- **stats**: 数据统计信息
  - conversations: 对话数量
  - messages: 消息总数
  - moments: 朋友圈数量
  - contacts: 联系人数量
  - documents: 文档数量
  - images: 图片数量
  - profiles: AI角色数量
  - relationships: 关系条数
- **data**: 所有实际数据
  - 键名为localStorage的key
  - 值为解析后的JSON对象或原始字符串

## 💡 使用建议

### 备份策略

1. **定期备份**
   - 建议每周导出一次数据
   - 重要对话后立即备份
   - 更新应用前备份

2. **多地备份**
   - 保存到本地硬盘
   - 上传到云盘（如iCloud、OneDrive）
   - 保存到U盘或移动硬盘

3. **版本管理**
   - 文件名包含日期，易于区分
   - 保留最近3-5个版本
   - 删除过旧的备份

### 迁移场景

1. **更换设备**
   - 旧设备导出数据
   - 新设备导入数据
   - 所有内容完整迁移

2. **重装应用**
   - 卸载前导出数据
   - 重装后导入数据
   - 恢复到之前状态

3. **多设备同步**
   - 主设备导出数据
   - 其他设备导入数据
   - 实现数据同步（手动）

### 注意事项

⚠️ **导入前必读**：
- 导入会**完全覆盖**当前数据
- 建议先导出当前数据作为备份
- 确认备份文件来源可靠
- 检查备份文件的日期和统计信息

⚠️ **数据安全**：
- 备份文件包含API Key等敏感信息
- 不要分享备份文件给他人
- 存储在安全的位置
- 考虑加密存储

## 🔧 技术实现

### 导出逻辑

```typescript
// 1. 遍历localStorage获取所有数据
for (let i = 0; i < localStorage.length; i++) {
  const key = localStorage.key(i);
  const value = localStorage.getItem(key);
  // 尝试解析JSON，失败则保存原始字符串
  allData[key] = JSON.parse(value) || value;
}

// 2. 统计各类数据量
// conversations, messages, contacts, documents等

// 3. 创建导出对象
const exportData = {
  exportDate: new Date().toISOString(),
  appVersion: '1.0.0',
  dataType: 'full-backup',
  stats: stats,
  data: allData
};

// 4. 生成JSON文件并下载
const blob = new Blob([JSON.stringify(exportData, null, 2)]);
// ... 创建下载链接
```

### 导入逻辑

```typescript
// 1. 读取JSON文件
const importedData = JSON.parse(fileContent);

// 2. 验证格式
if (!importedData.data || typeof importedData.data !== 'object') {
  throw new Error('Invalid format');
}

// 3. 显示预览和确认
// 显示stats统计信息，请求用户确认

// 4. 清空localStorage
localStorage.clear();

// 5. 恢复所有数据
for (const key in importedData.data) {
  const value = importedData.data[key];
  localStorage.setItem(key, 
    typeof value === 'object' ? JSON.stringify(value) : value
  );
}

// 6. 刷新页面
window.location.reload();
```
