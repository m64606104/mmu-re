## 📊 技术架构

### 主屏幕实现
```typescript
// 滑动检测
onTouchStart → 记录起始位置
onTouchMove → 记录移动位置
onTouchEnd → 计算滑动距离，切换页面

// 页面切换
transform: translateX(-${currentPage * 100}%)
transition: 300ms ease-out

// 页面指示器
当前页：w-6 bg-gray-600
其他页：w-1.5 bg-gray-300
```

### AI朋友圈实现
```typescript
// 周期管理
getMomentCycleData() → 获取周期数据
shouldTriggerAIMoment() → 判断是否发布
incrementPostCount() → 增加计数
resetMomentCycle() → 重置周期

// 内容生成
getTimeContext() → 获取时间场景
getRecentChatContext() → 获取聊天内容
generateAIMomentContent() → 生成朋友圈
```
