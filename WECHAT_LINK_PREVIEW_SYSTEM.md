# 微信风格链接预览系统

## 📱 系统概述

全新的微信风格链接预览系统，取代原有生硬的文档卡片，模仿真实微信中打开小红书、知乎、微博等链接的自然展示效果。

---

## 🎯 核心改进

### **Before（旧系统）**
```
❌ 硬编码的文档卡片
❌ 刻意的"请查收·在线文档"标签
```

```
✅ 支持多种自然格式
```

## 📦 新增组件

### **1. WeChatLinkPreview.tsx**
微信风格链接预览卡片组件

**设计特点：**
- 白色背景 + 细灰边框
- 标题 + 描述 + 封面图（可选）
- 底部平台标识（小图标 + 平台名称）
- 点击可展开查看完整内容

**支持平台：**
- 📕 小红书 (xiaohongshu)
- 💡 知乎 (zhihu)
- 🎭 微博 (weibo)
- 📱 微信公众号 (wechat)
- 📰 新闻 (news)
- 📄 文档 (document)
- 🌐 网页 (web)

---

### **2. SmartLinkParser.ts**
智能链接解析器

**功能：**
1. 从AI自然文本中提取链接预览信息
2. 支持7种格式（从精确JSON到自然标记）
3. 自动去除标记，保留纯文本
4. 兼容旧的[发文档]格式

**支持格式：**

#### **格式1：JSON格式（最精确）**
```
我给你推荐一篇文章 {"type":"link","platform":"xiaohongshu","title":"秋日穿搭分享","description":"分享了5套温柔风穿搭","coverImage":"一个女生穿着米色毛衣"}
```

#### **格式2：简化标记（推荐）**
```
[链接:小红书:秋日穿搭分享:分享了5套温柔风穿搭:一个女生穿着米色毛衣]
```

#### **格式3：平台专用格式**
```
小红书：[小红书:标题:描述:封面]
知乎：[知乎:标题:描述]
微博：[微博:标题:描述:作者]
公众号：[公众号:标题:描述:公众号名]
```

#### **格式4：兼容旧文档格式**
```
[发文档:标题:类型] 内容...
→ 自动转换为document平台的链接预览
```

## 💻 集成说明

### **类型定义（types.ts）**
```typescript
// 新增LinkPreviewMessage类型
export interface LinkPreviewMessage {
  title: string;
  description?: string;
  coverImage?: string;
  platform: 'xiaohongshu' | 'zhihu' | 'weibo' | 'wechat' | 'news' | 'web' | 'document';
  url?: string;
  author?: string;
  publishTime?: string;
  content?: string;  // 完整内容，点击后展开
}

// Message接口新增linkPreview字段
export interface Message {
  // ... 其他字段
  linkPreview?: LinkPreviewMessage;  // 新增
  document?: DocumentMessage;  // 保留兼容
}
```

### **AI消息解析（ChatScreen.tsx）**
```typescript
// 在AI消息处理中集成
const parsedLink = SmartLinkParser.parseMessage(finalContent);
if (parsedLink.linkPreviews.length > 0) {
  // 为每个链接创建消息
  parsedLink.linkPreviews.forEach((linkPreview, idx) => {
    allExtraMessages.push({
      id: `${baseId}_link_${idx}`,
      role: 'assistant',
      content: `分享了链接`,
      linkPreview
    });
  });
  
  // 更新为纯文本
  finalContent = parsedLink.textContent;
}
```

### **消息显示（ChatScreen.tsx）**
```typescript
{/* 新系统：微信风格链接预览 */}
{message.linkPreview && (
  <WeChatLinkPreview
    data={message.linkPreview}
    onClick={() => {
      if (message.linkPreview!.content) {
        setViewingDocument({
          title: message.linkPreview!.title,
          content: message.linkPreview!.content,
          type: 'text'
        });
      }
    }}
  />
)}

{/* 旧系统：保留兼容 */}
{!message.linkPreview && message.document && (
  <DocumentCard ... />
)}
```
