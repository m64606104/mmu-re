### 2. **支持的文档格式**

#### ① JSON 格式（最高优先级）
```json
{
  "title": "失控",
  "content": "<p>内容...</p>",
  "type": "text",
  "greeting": "请查收"
}
```

#### ② HTML <doc> 标签
```html
<doc title="失控" doc-type="text">
  <p>内容...</p>
</doc>
```

#### ③ Markdown 格式
```markdown
# 失控

这是正文...
```

#### ④ 自然语言 ⭐️ **新增**
```
发送了文档《失控》
这是正文...
```

#### ⑤ 旧版标记（兼容）
```
[发文档:失控:text] 这是正文...
```
