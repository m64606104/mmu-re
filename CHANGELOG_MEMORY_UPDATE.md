## 🚀 使用方法

### 设置角色网名
1. 进入角色设置
2. 填写"角色网名"字段
3. 保存

### 使用记忆库
1. 进入角色设置
2. 点击"查看长期记忆库"
3. 启用自动记忆总结
4. 设置总结间隔
5. 可手动添加记忆
