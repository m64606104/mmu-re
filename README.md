# 移动 AI 聊天应用

一个模拟手机界面的 AI 聊天网页应用，支持私聊和群聊功能。

## 功能特性

- 📱 **手机界面模拟** - 仿真手机屏幕设计，包含主屏幕、应用图标
- 💬 **私聊功能** - 与 AI 角色进行一对一对话
- 👥 **群聊功能** - 多人对话场景模拟
- ⚙️ **API 设置** - 支持自定义 OpenAI 兼容 API 配置
- 🎭 **角色设置** - 为私聊角色自定义头像、性格、语言风格等
- 💾 **本地存储** - 所有对话和设置保存在浏览器本地

## 技术栈

- React 18
- TypeScript
- Vite
- TailwindCSS
- Lucide React (图标)

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 使用说明

### 1. 配置 API

1. 在主屏幕点击"设置"应用
2. 填写 Base URL（如：https://api520.pro）
3. 填写 API Key
4. 点击"测试连接"自动获取可用模型
5. 选择模型后点击"保存配置"

### 2. 创建对话

1. 在主屏幕点击"社交"应用
2. 点击右上角"+"按钮
3. 选择对话类型（私聊/群聊）
4. 输入名称并创建

### 3. 聊天功能

- **输入消息**：在底部输入框输入文本，按回车或点击发送按钮添加到待发送列表
- **发送消息**：点击"发送消息"按钮将所有待发送消息发送到对话中
- **生成回复**：点击"生成回复"按钮，AI 将根据对话历史生成回复
- **多条回复**：AI 可以一次生成多条消息

### 4. 角色设置（仅私聊）

在私聊界面点击右上角"角色设置"，可以自定义：

- **角色头像**：上传自定义头像图片
- **备注名**：修改角色显示名称
- **人物设定**：描述角色背景、身份等
- **性格特征**：定义角色性格
- **语言风格**：设置说话方式
- **语言示例**：提供对话示例
- **记忆事件**：记录重要事件

## API 兼容性

本应用使用 OpenAI 兼容的 API 格式，支持任何遵循 OpenAI API 规范的服务，包括：

- OpenAI 官方 API
- api520.pro
- 其他兼容服务

## 数据存储

所有数据（对话记录、API 配置、角色设置）都保存在浏览器的 localStorage 中，不会上传到任何服务器。
