# 词卡生成问题修复报告

## 🚨 发现的问题

### 1. 词卡数量错误
- **问题**: 生成60个词而不是期望的20个词
- **影响**: UI显示"今日 20/20"但实际生成60个词，造成数量不匹配
- **原因**: 代码设计为15轮×4词=60词，但用户期望是5轮×4词=20词

### 2. require错误
- **问题**: `ReferenceError: require is not defined`
- **影响**: 教学失败时出现JavaScript错误
- **原因**: 在浏览器环境中使用了Node.js的`require`函数

## 🔧 修复方案

### 1. 词卡数量修复

**生成函数修改**:
```typescript
/**
 * 为当天生成5轮词卡（共20个词，符合用户期望）
 */
export async function generateDailyCards(...) {
  // 使用AI生成20个适合的词（5轮×4词）
  const words = await generateWordsWithAI(...);
  
  // 分成5轮，每轮4个词
  const rounds: WordCard[][] = [];
  for (let i = 0; i < 5; i++) {
    rounds.push(words.slice(i * 4, (i + 1) * 4));
  }
}
```

### 2. require错误修复

#### 问题定位
在以下文件中发现了`require`调用：
- `src/components/CharacterSettingsScreen.tsx:333`
- `src/utils/aiKindergartenManager.ts:301, 331, 460`

#### 修复方案
将所有`require`调用替换为动态`import()`：

**CharacterSettingsScreen.tsx**:
```typescript
// 修复前
const { saveDocument } = require('../utils/documentLibrary');

// 修复后
const { saveDocument } = await import('../utils/documentLibrary');
```

**aiKindergartenManager.ts**:
```typescript
// 修复前
const { updateChildComprehension } = require('./correctComprehensionSystem');

// 修复后
const { updateChildComprehension } = await import('./correctComprehensionSystem');
```

**异步函数更新**:
```typescript
// 修复前
export const updateComprehension = (childData: AIChildData): void => {
  const { recalculateComprehensionFromVocabulary } = require('./correctComprehensionSystem');
  recalculateComprehensionFromVocabulary(childData);
};

// 修复后
export const updateComprehension = async (childData: AIChildData): Promise<void> => {
  const { recalculateComprehensionFromVocabulary } = await import('./correctComprehensionSystem');
  recalculateComprehensionFromVocabulary(childData);
};
```

**调用点更新**:
```typescript
// 修复前
updateComprehension(conversation.aiChildData);

// 修复后
await updateComprehension(conversation.aiChildData);
```
