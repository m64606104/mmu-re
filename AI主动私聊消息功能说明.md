# AI主动私聊消息功能说明

## 🎯 功能概述

当AI在群聊中承诺要私聊发送内容给用户时（如"我私聊发给你搞笑视频"），系统会自动检测到这个承诺，并在几秒后主动在对应的私聊会话中发送相关内容。

## 🔧 工作原理

### 1. 承诺检测
系统会检测AI消息中是否包含以下关键词：
- `我私聊发给你`、`我私下发给你`、`我单独发给你`
- `私聊发`、`私下发`、`单独发`
- `我私聊给你发`、`我私下给你发`、`我单独给你发`
- `等下私聊发`、`等下私下发`、`等下单独发`
- `稍后私聊发`、`稍后私下发`、`稍后单独发`

### 2. 私聊会话查找
使用AI的ID（`message.senderId`）查找对应的私聊会话。

### 3. 延迟发送
延迟3-8秒后发送私聊消息，模拟真实行为。

### 4. 内容生成
调用Chat API生成符合承诺的内容，支持：
- `[视频:描述]` 格式的视频内容
- `[图片:描述]` 格式的图片内容
- 有用的链接
- 其他符合承诺的内容

## 📝 使用示例

### 群聊场景
```
用户: 有人有好玩的视频分享吗？
AI: 我私聊发给你一些搞笑的视频，等下哈
```

### 私聊结果（几秒后自动发送）
```
AI: 哈哈，说到搞笑视频，我私聊发给你一个超萌的猫咪视频！[视频:一只橘猫在追激光笔，突然撞到沙发上的搞笑瞬间] 刚才在群里人多不好意思发，这个真的太逗了，你看了一定会笑！
```

## 🔍 技术细节

### 核心文件
- `src/components/ChatScreen.tsx` - 主要实现逻辑
- `src/utils/autoPrivateDMTest.ts` - 测试工具
- `src/utils/groupToPrivateMemoryService.ts` - 记忆桥接服务

### 关键函数
- `maybeSendAutoPrivateDM()` - 主动发送私聊消息的主函数
- `detectPrivatePromise()` - 检测承诺关键词
- `generatePrivateContent()` - 生成私聊内容

### API调用
使用标准的OpenAI Chat API，请求体结构：
```json
{
  "model": "gpt-3.5-turbo",
  "messages": [
    {
      "role": "system",
      "content": "你是AI助手名称。系统提示词..."
    },
    {
      "role": "user", 
      "content": "你在群聊中承诺要私聊发送内容给用户..."
    }
  ],
  "temperature": 0.7,
  "max_tokens": 500
}
```

## 🎯 限制和注意事项

1. **关键词匹配**：目前基于关键词匹配，可能无法识别所有表达方式
2. **API依赖**：需要有效的OpenAI API密钥
3. **延迟机制**：固定3-8秒延迟，可能不适合所有场景
4. **内容质量**：生成的内容质量取决于AI模型能力

## 🚀 未来改进方向

1. **智能检测**：使用NLP技术更准确地识别承诺意图
2. **个性化延迟**：根据承诺内容类型调整延迟时间
3. **多媒体支持**：支持真实的图片、视频文件发送
4. **用户偏好**：记住用户对不同类型内容的偏好
