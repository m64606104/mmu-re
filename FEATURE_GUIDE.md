## 📖 使用指南

### 💰 红包转账功能

#### 发送红包（用户）
1. 打开聊天 → 点击 **+** 按钮
2. 点击 **💰** 红包按钮
3. 选择 **红包** 或 **转账**
4. 输入金额和留言
5. 点击发送

#### AI自动处理
- AI收到红包后会自动判断是否接收
- 根据关系亲密度和金额大小决定
- 亲密+合理金额 → 接收并感谢
- 疏远+金额过大 → 礼貌退回

#### AI主动发红包
AI会在以下场景主动发红包：
```
场景1：生日祝福
用户：今天我生日
AI：  生日快乐！🎂 [发红包:66.6:祝你天天开心]

场景2：表达感谢
用户：谢谢你帮我
AI：  不客气！ [发红包:8.8:小小心意]

场景3：节日祝福
用户：中秋节快乐
AI：  中秋快乐！ [发红包:88:中秋团圆]
```

#### AI主动转账
AI会在以下场景主动转账：
```
场景1：还钱
用户：上次借你100块
AI：  哦对，我转给你 [转账:100:还钱]

场景2：AA付款
用户：昨天吃饭你帮我付了
AI：  我转给你我那份 [转账:50:昨天的饭钱]
```

### 📄 文档功能

#### 发送文档（用户）
1. 打开聊天 → 点击 **+** 按钮
2. 点击 **📄** 文档按钮
3. 输入文档标题
4. 选择文档类型（text/markdown/code）
5. 输入文档内容
6. 可选：添加问候语
7. 点击发送

#### 查看文档
- 点击文档卡片 → 打开查看器
- 支持滚动阅读
- 点击返回按钮关闭

#### AI主动发送文档
AI会在以下场景主动发送文档：
```
场景1：写信
用户：能写封情书吗
AI：  我写好了 [发文档:给你的一封信:text]
     
     亲爱的朋友：
     
     见字如面...
     
     落款：你的朋友

场景2：分享代码
用户：帮我写个冒泡排序
AI：  这是代码 [发文档:冒泡排序:code]
     
     function bubbleSort(arr) {
       for (let i = 0; i < arr.length; i++) {
         for (let j = 0; j < arr.length - 1 - i; j++) {
           if (arr[j] > arr[j + 1]) {
             [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
           }
         }
       }
       return arr;
     }

场景3：整理笔记
用户：帮我总结一下今天的会议
AI：  我整理好了 [发文档:会议纪要:markdown]
     
     # 会议纪要
     
     ## 时间：2024-XX-XX
     
     ## 讨论内容：
     ...
```

### 💳 钱包功能

#### 查看钱包
1. 点击 **"我"** 标签
2. 点击 **💰 钱包**
3. 查看余额和交易记录

#### 购物功能
1. 钱包页面 → 点击购物服务
2. 选择商店（饿饿吗/电影票/淘淘宝）
3. 浏览商品或搜索
4. 点击购物车购买
5. 自动扣款

#### AI生图购物
1. 购物页面 → 点击 **⚙️** 设置
2. 填写AI生图API配置
   - API URL（如：https://api.openai.com/v1/images/generations）
   - API Key
3. 保存配置
4. 搜索商品（如"想吃火锅"）
5. AI自动生成商品图片
6. 点击购买

## 🤖 AI指令格式

### 红包转账
```
发红包：[发红包:金额:留言]
转账：  [转账:金额:备注]

示例：
- [发红包:66.6:生日快乐]
- [转账:100:还钱]
```

### 文档发送
```
[发文档:标题:类型]

文档内容...

落款：xxx

类型：text | markdown | code

示例：
- [发文档:情书:text]
- [发文档:代码示例:code]
- [发文档:笔记:markdown]
```

## 💡 使用技巧

### 红包技巧
1. **吉利数字**：6.6、8.8、66.6、88.8、188
2. **整数转账**：还钱用整数（100、200）
3. **留言简短**：不超过20字

### 文档技巧
1. **标题简洁**：不超过50字
2. **内容分段**：使用换行分段
3. **代码文档**：使用等宽字体
4. **markdown**：支持简单段落换行

### AI互动技巧
1. **自然对话**：像真人一样聊天，AI会自动判断场景
2. **明确需求**：直接说"写封信"、"发个红包"
3. **情感表达**：生日、感谢等场景更容易触发AI发红包

## 🐛 故障排查

### 红包无法发送
- ✅ 检查余额是否足够
- ✅ 金额是否大于0
- ✅ 钱包初始余额1000元

### AI不发红包
- ✅ 场景是否合适（生日、感谢等）
- ✅ API配置是否正确
- ✅ 尝试更明确的表达

### 文档无法查看
- ✅ 检查文档是否发送成功
- ✅ 点击文档卡片打开
- ✅ 刷新页面重试

### AI生图失败
- ✅ 检查API URL是否正确
- ✅ 检查API Key是否有效
- ✅ 检查API配额是否用完
