### 🔄 自由模式多轮回复优化

#### 当前问题
自由模式下，每个AI在一轮生成中只能回复一次，不够自然。

#### 优化方向

**方案1: 多轮循环**
```
第1轮: AI随机选择是否回复
第2轮: 基于第1轮的回复，AI再次决定
第3轮: 继续...
直到: 没有AI回复或达到最大轮数
```

**方案2: 对话热度机制**
```
热度值: 基于话题有趣度和参与度
高热度: AI更倾向于继续参与
低热度: AI倾向于结束对话
```

**方案3: 自然结束检测**
```
检测对话是否已经自然结束
- 问题已回答
- 话题已完结
- 大家都说完了
```

#### 建议实现
结合方案1和方案3：
1. 设置最大3轮循环
2. 每轮后检测是否自然结束
3. 如果话题还在继续，允许AI再次回复
4. 避免无限循环

#### 实现要点
```typescript
async function generateGroupChatRepliesFreeMode() {
  let round = 0;
  const maxRounds = 3;
  let hasNewReplies = true;
  
  while (round < maxRounds && hasNewReplies) {
    round++;
    const roundReplies = await generateRound();
    
    if (roundReplies.length === 0) {
      hasNewReplies = false;
    } else {
      // 检测是否自然结束
      const shouldContinue = detectConversationContinuation(roundReplies);
      if (!shouldContinue) {
        hasNewReplies = false;
      }
    }
  }
}
```

- `src/utils/groupChatService.ts` - 修改自由模式逻辑
- 可能需要新增对话分析函数
