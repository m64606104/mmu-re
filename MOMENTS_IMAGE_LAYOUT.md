## 💻 代码实现

### CSS类名映射

```typescript
const getImageGridClass = (count: number) => {
  switch (count) {
    case 1:
      return 'grid-cols-1 max-w-[200px]'; // 单图：单独一列
    case 2:
      return 'grid-cols-2 gap-1'; // 2张：横向2列
    case 3:
      return 'grid-cols-3 gap-1'; // 3张：横向3列
    case 4:
      return 'grid-cols-2 gap-1'; // 4张：2x2网格
    default:
      return 'grid-cols-3 gap-1'; // 5-9张：3列布局
  }
};
```

### 图片尺寸类名

```typescript
const getImageItemClass = (count: number) => {
  if (count === 1) {
    return 'aspect-[4/3]'; // 单图：4:3比例
  }
  return 'aspect-square'; // 其他情况：正方形
};
```

## 🔧 技术细节

### Tailwind CSS类

| 数量 | grid-cols | aspect | 说明 |
|------|-----------|--------|------|
| 1 | grid-cols-1 | aspect-[4/3] | 单独一列，4:3比例 |
| 2 | grid-cols-2 | aspect-square | 2列，正方形 |
| 3 | grid-cols-3 | aspect-square | 3列，正方形 |
| 4 | grid-cols-2 | aspect-square | 2列，2行 |
| 5-9 | grid-cols-3 | aspect-square | 3列，自动行数 |

### gap间距
```css
gap-1  /* 0.25rem = 4px */
```

### 单图最大宽度
```css
max-w-[200px]  /* 限制单图最大宽度 */
```
